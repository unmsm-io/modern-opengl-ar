## Clonar el proyecto

```bash
git clone https://github.com/FISI-SM/locked-in-cg.git
cd locked-in-cg
```

## Instalar Homebrew y paquetes

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install glfw glm opencv pkg-config
```

*Glad ya viene vendorizado en `external/glad`, no uses brew.*
